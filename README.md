# dashbord
## Pravane Lavani Sales Dashboard

### 📌 Project Overview

A simple and interactive **Power BI Sales Dashboard** created to analyze sales performance, profit, quantity sold, and discounts. The dashboard helps users understand business performance through visual reports and key metrics.

### 🎯 Objective

* Monitor sales and profit performance.
* Analyze sales by category, region, and customer segment.
* Identify business trends and top-performing areas.
* Support better decision-making using data visualization.

### 🛠️ Tech Stack

* Power BI Desktop
* Microsoft Excel
* Power Query
* DAX (Data Analysis Expressions)

### 📊 Dashboard Features

* Total Sales, Profit, Quantity, and Discount KPIs
* Sales by Sub-Category
* Sales by Region
* Sales by Customer Segment
* State-wise Filtering
* Interactive Charts and Visualizations

### 📂 Data Source

Excel sales dataset containing:

* Order Details
* Product Information
* Customer Segments
* Region & State Details
* Sales, Profit, Quantity, and Discount Data

### 📈 Key Insights

* Consumer segment generates the highest sales.
* Sales performance varies across regions and states.
* Some product categories contribute more revenue than others.
* Interactive filters allow detailed business analysis.

### ✅ Outcome

Developed an interactive dashboard that converts raw sales data into meaningful insights, helping businesses track performance and make data-driven decisions.
